This is a web-based wallet for ethereum and erc20 ethereum-based tokens. To proceed:

  • If you don't have a wallet, create one using Create New Wallet section. Save your address and private key since we don't store anything.
  • If you already have a wallet (an account in methamask or myetherwallet, etc.) you can use your private key to unlock your account using Unlock Wallet and check you balance or see the transaction history.
  • If you want to see information about a specific token you have in your account, or you want to transfer that custom token, use the token contract address in Add Custom Token section.
  • Use the last two sections Send Ether and Send Token to transfer ether/token from the unlocked account to any other address.

This is a test wallet for learning purpose and I do not accept any resposibility for possible issues!

Do not use this page with a browser that has metamask extention or use it in incognito mode.

Everything in this page is on ropsten test net.
